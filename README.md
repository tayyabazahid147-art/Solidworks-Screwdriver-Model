# Solidworks-Screwdriver-Model
This project demonstrates the design of a screwdriver using SOLIDWORKS.  The model was created as a CAD practice project to improve parametric modeling skills.
## Software Used

- SOLIDWORKS



## Skills Demonstrated

- 2D Sketching
- Extruded Boss/Base
- Revolved Features
- Fillet
- Parametric Design
- Feature Tree Management
